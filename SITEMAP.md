# Sitemap: Portfolio Transformation Platform

## 1. Public / Landing Zone
*   **Home Page:** Value proposition, "Before vs. After" AI transformation examples.
*   **Pricing:** Tiered access (Free, Pro, Strategic).
*   **Case Studies:** Success stories of creators getting hired.

## 2. User Workspace (The Dashboard)
*   **Overview:** High-level stats (Portfolio views, Match score trends).
*   **My Assets:**
    *   **Project Vault:** Repository of all raw projects/case studies.
    *   **Skill Graph:** Interactive visualization of the user's AI-extracted skills.

## 3. Transformation Engine (Core Workflow)
*   **Optimizer:**
    *   **Narrative Lab:** AI-assisted editor for project storytelling.
    *   **Impact Quantifier:** Tool to turn bullet points into data-driven results.
*   **Market Intelligence:**
    *   **Targeting Tool:** Paste JD -> Get Match Score & Suggestions.
    *   **Trend Watch:** Insights on trending skills in specific creative niches.

## 4. Showcase & Distribution
*   **Link Manager:** Create and manage "Custom Views" for specific leads.
    *   **View Settings:** Password protection, expiration dates, custom branding.
*   **Analytics:** Heatmaps and "Time on Project" data for shared links.

## 5. Account & Settings
*   **Profile:** Personal branding, social links, resume upload.
*   **Integration:** Connect Google Drive, Behance, Dribbble, or GitHub for data import.
*   **Billing:** Subscription management.

---

## Navigation Logic
*   **Primary Navigation:** Dashboard, Projects, Market Insights, Showcases.
*   **Contextual Actions:** "Optimize this project" (links Project Vault to Narrative Lab), "Match with Job" (links Project Vault to Targeting Tool).
